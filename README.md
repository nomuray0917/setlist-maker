# Setlist Maker

バンド活動やライブイベントのためのセットリスト作成・管理ツールです。
曲順の構成、演奏時間の計算、そして本番用のPDF出力までをこのアプリひとつで完結できます。

## 概要

実際のバンド活動（carrel bitesなど）での利用を想定して開発されたデスクトップアプリケーションです。
GUIで直感的にセットリストを組み、リハーサルや本番でスタッフ・メンバーに配布可能なA4サイズのPDFを即座に生成します。

## 主な機能

* **セットリスト作成**: 曲名、演奏時間、照明や機材に関する備考を入力可能。
* **MC対応**: 曲だけでなく、MC（トーク）のパートも視覚的に分かりやすく挿入。
* **PDF出力**: ボタン一つで、整ったレイアウトのPDFファイルを生成（`export`フォルダに出力）。
* **ファイル保存**: `.set` 形式（JSONベース）でプロジェクトを保存・読み込み可能。
* **マルチバンド対応**: 複数のバンドを掛け持ちしている場合でも、アーティスト名を切り替えて管理可能。
* **演奏時間計算**: 曲ごとの時間を入力すると、セットリスト全体の合計時間を自動計算（ON/OFF切替可能）。
* **クリップボードコピー**: LINEなどのチャットツールに貼り付けやすいテキスト形式でコピー。
* **安全設計**: 未保存の変更がある場合に警告を表示し、データの消失を防止。

## ダウンロードと起動方法

このアプリケーションは、インストール不要の `.exe` ファイルとして配布しています。

1.  このリポジトリの **[Releases]** ページを開いてください。
2.  最新バージョンの `SetlistMaker.exe` をダウンロードしてください。
3.  ダウンロードしたファイルを任意の場所（デスクトップなど）に置き、ダブルクリックして起動してください。
4.  初回起動時、自動的に保存用フォルダ（`setlists`）と出力用フォルダ（`export`）が作成されます。

> **※ Windows Defender等の警告が出る場合**
> 署名のない個人開発アプリのため、起動時に「WindowsによってPCが保護されました」という警告が表示されることがあります。
> その場合は、**「詳細情報」** をクリックし、**「実行」** ボタンを押してください。

## 使い方

1.  **基本情報の入力**:
    * 画面左上のプルダウンで「バンド名」を選択（「バンド管理」から追加可能）。
    * 「日付」「イベント名」を入力します。
2.  **曲の追加**:
    * 曲名を入力し、`Enter` または「曲を追加」ボタンを押します（`Ctrl + Enter` ショートカット対応）。
    * 「演奏時間を有効にする」にチェックを入れると、時間の管理ができます。
3.  **MCの追加**:
    * 「MCを追加」ボタンでMCパートを挿入します。
4.  **リストの編集**:
    * リストを選択し、「▲ 上へ」「▼ 下へ」ボタンで曲順を変更します。
    * 削除したい場合は「削除」ボタンを使用します。
5.  **保存と出力**:
    * **保存**: `.set` ファイルとして保存し、後で再編集できます。
    * **PDF書き出し**: A4サイズのPDFを出力します。印刷してステージで使用できます。
